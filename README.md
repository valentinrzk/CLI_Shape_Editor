# Что это?
Простейший CLI редактор для CRUD операций с геометрическими фигурами.
# Что умеет?
Добавлять фигуры(точка, линии, окружность, квадрат), удалять, показывать списком, вычислять площадь и периметр.
Рендерить не умеет, только хранить данные :(
# Установка и использование
1. Клонируем репозиторий: `git clone https://github.com/valentinrzk/CLI_Shape_Editor.git`
2. Переходим в папку проекта: `cd <папка_проекта>`
3. Создаем виртуальное окружение: `python -m venv venv`
4. Устанавливаем зависимости: `pip install -r requirements.txt`. В данном случае зависимостей нет, только pytest, так что шаг опционален.
4. Запускаем программу: `python main.py`
# Мейн юзер флоу
1. Запускаем программу в терминале.
2. Пишем команду для добавления фигуры, например: "add circle 5" (добавить окружность с радиусом 5).
3. Пишем команду для отображения всех фигур: "list".
4. Пишем команду для вычисления площади фигуры, например: "area 1" (вычислить площадь фигуры с ID 1).
5. Пишем команду для удаления фигуры, например: "delete 1" (удалить фигуру с ID 1).
6. Пишем команду help для отображения списка доступных команд и их синтаксиса, играемся еще, пока не надоест.
6. Пишем "exit" для выхода из программы.
# Архитектурная идея
1. Хранилище - in memory
2. Обработка команд - через реестр команд. Новые команды добавляются в реестр через декоратор.
3. Фигуры - наследуются от базового класса, создание через фабрику, добавление новых видов фигур в реестр через декоратор.
# Тестирование
1. Установить `pip install -r requirements.txt`, если не сделали этого ранее.
2. Запустить тесты: `pytest -v`
15 тестов, покрытие 100% но без фанатизма. Есть фикстуры и параметризация.
