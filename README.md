# CLI Векторный редактор

Простой интерактивный редактор для работы с геометрическими фигурами через командную строку.

## Возможности

- ✅ Создание фигур (точка, линия, окружность, квадрат, прямоугольник, овал)
- ✅ Удаление и очистка фигур
- ✅ Просмотр списка всех фигур
- ✅ Вычисление площади и периметра
- ✅ Сохранение и загрузка в JSON

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/valentinrzk/CLI_Shape_Editor.git
cd CLI_Shape_Editor

# Создать виртуальное окружение
python -m venv venv

# Установить зависимости
pip install -r requirements.txt

# Запустить программу
python main.py
```
# Команды
```bash
- help       - Показать список доступных команд. Информация о командах берется из их docstring, а также из атрибута syntax для команд создания фигур.
- add        - Создать фигуру. Подробности создания фигур см. в разделе "Синтаксис создания фигур"
- list       - Команда вывода списка фигур.
- delete     - Команда удаления фигуры.
- area       - Команда вычисления площади фигуры.
- perimeter  - Команда вычисления периметра фигуры.
- clear      - Команда очистки всех фигур.
- save       - Сохранить фигуры в файл.
- load       - Загрузить фигуры из файла.
```

## Синтаксис создания фигур:

```bash
- add point <x> <y>              Создать точку с координатами x и y.
- add line <x1> <y1> <x2> <y2>   Создать линию с началом x1 y1 и концом x2 y2.
- add circle <x> <y> <r>         Создать окружность с координатами x y, а также радиусом r.
- add square <x> <y> <side>      Создать квадрат с координатами x y, а также длиной стороны side.
- add rectangle <x> <y> <width> <height> Создать прямоугольник с координатами x y, а также шириной width и высотой height.
- add ellipse <x> <y> <xr> <yr>  Создать овал с координатами x y, а также радиусами xr по оси X и yr по оси Y.
```

# Архитектура
- Хранилище — in-memory хранилище фигур
- Команды — реестр команд с декоратором регистрации
- Фигуры — иерархия классов с наследованием от базового класса
- Фабрика — паттерн Factory для создания фигур
- Файловый сервис — сохранение и загрузка в JSON
